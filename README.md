# Getting Started

1. Open the project with Unity 6
2. Open the 0-Main.unity scene
3. Press Play

# Card Creator Tool

Scenes/CardCreator.unity is the scene that would generate teh 144 card sprites. The output files are saved in `Sprites/Cards/card-xxx.png`

1. Load the CardCreator scene
2. Press Play. The cards will automatically be generated.
